diz o que tal parte do formulário é for=""; pode relacionar com o id do input. entrada de dados. type=""; define o tipo do dado digitado. email; usado para emails(semântico). radio; marcar bolinha. submit; envia as informações. button; faz um botão. reset; reseta as informações. number; aceita apenas números. text; aceita qualquer tipo de texto. date; datas com calendário. range; permite o controle de algo com números min e max. url; aceita urls. time; faz um relógio. color; cores. checkbox; marcar caixinha. checked; checa automaticamente ao abrir a janela. file; aceita arquivos do pc. accept=""; tipos de arquivo aceitos. hidden; esconde informações do usuário. tel; telefone. name=""; da um nome pros valores das tags. É necessário ter o mesmo nome no back-end. id=""; usado pra relacionar o input com uma label. value=""; valor enviado para o servidor ou para ser selecionado. min="" e max=""; valores mínimo e máximo. autofocus; foco automático ao abrir a janela. required; não da para prosseguir até cumprir o requisito. maxlength; comprimento máximo. step; escolhe de quanto em quanto os números avançam(0.2|0.5|2). pattern; cria um padrão para a digitação([0-9]{11}). placeholder; dica pro usuário digitar. size; define o limite máximo em números(size="10"). multiple; pode selecionar mais de uma. cols=""; número de colunas. rows=""; número de linhas. readonly; mesma aparência porém não é possível editar. disabled; desabilita a caixa para edições e muda a cor. onclick=""; acontece algo quando clica. Pode ser usado junto com JS. atributos para ambos: controls; exibe os controles. autoplay; assim que a página carrega começa a tocar.
